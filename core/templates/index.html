Sobre

A Ferrugem Games é uma idéia de loja virtual. Tendo como inspiração as grandes lojas onlines hoje já disponíveis, como Steam, Green Man Gaming e GOG.com.

Esse é um projeto em grupo da Universidade Anhembi Morumbi, matéria disciplinar Usabilidade, Desenvolvimento e Web. A ideia é fazermos uma página Web responsiva utilizando HTML, CSS e Javascript.
